INSTALLATION                                         *aiassistant-installation*


Use your favourite package manager, or use Vim's built-in package support.

Vim:~
>
  mkdir -p ~/.vim/pack/aclaro/start
  cd ~/.vim/pack/aclaro/start
  git clone https://github.com/alejandroclaro/vim-ai-assistant
  vim -u NONE -c "helptags vim-ai-assistant/doc" -c q
<
